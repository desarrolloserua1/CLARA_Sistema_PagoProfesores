Ayuda: Importación de PA desde Banner

Esta pantalla se utiliza para importar la Programación Academica de Banner al Sistema de Pago a Profesores, la importación se puede realizar por medio de varios filtros a elegir:

Periodo, Parte de Periodo, Escuela, Tipos de Pago, Campus

Este proceso importa la información de banner en el sistema pago a profesores, los datos que se importan se muestran a continuación:

Datos Cursos a Importar

  • ID SIU
  • Docente (Nombre)
  • Docente (Apellidos)
  • Campus
  • Escuela
  • Nrc
  • Materia
  • Curso
  • Nombre de la Materia
  • Fecha Inicio Curso
  • Fecha Fin Curso
  • Tipo de Curso
  • Metodo de Instrucción
  • Status
  • Inscritos
  • Periodo
  • Parte del Periodo
  • Descripción del Periodo
  • Horas Programadas
  • % de Responsabilidad
  • Horas a Pagar
  • Opción de Pago
  • Tabulador

  • Docente (Nombre)
  • Docente (Apellidos)

  • CURP
